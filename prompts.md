# Prompt Engineering Process

## Prompt Used

Act as an SEO content strategist and professional blog writer.

Create an SEO-optimized blog for a dental clinic.

Requirements:
- Generate a blog outline with H1, H2, and H3 headings
- Write a detailed SEO-friendly blog
- Use keywords naturally
- Include local SEO references
- Suggest related content cluster topics
- Keep content informative and readable

Output should be suitable for a business website.
